# Combined Review Config

This file stores your project-specific settings for the Combined Review plugin.

## How to customize

1. Copy this file to your project root: `.claude/combined-review.local.md`
2. Edit the YAML frontmatter values above
3. The plugin will automatically read your config on next run

## Language options

| Value | Language |
|-------|----------|
| `system` | Auto-detect from CLAUDE.md or system locale |
| `en` | English |
| `ru` | Russian |
